## 🚀 Getting Started

### Coding Problems

Navigate to the `Problems` folder and run any Python script:
```sh
cd Problems
python caesar_cipher.py
```

## 📝 Project Details

- **Backend:** Node.js, Express, SQLite
- **Frontend:** React
- **Problems:** Python scripts for algorithmic challenges
